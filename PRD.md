# Product Requirements Document (PRD) - Prompt Architect

## 1. 프로젝트 개요
사용자가 AI를 비즈니스 업무에 더 효과적으로 활용할 수 있도록 돕는 "프롬프트 라이브러리 및 최적화 도구"입니다.

## 2. 목표 (Goals)
- 프롬프트 작성에 익숙하지 않은 사용자에게 고품질의 템플릿 제공.
- 단순한 요청을 AI를 통해 전문적인 지시서(Prompt)로 정교화.
- 직무별 세분화된 카테고리를 통해 업무 생산성 극대화.

## 3. 핵심 기능 (Core Features)

### 3.1. 프롬프트 라이브러리 (Prompt Library)
- **카테고리**: 보고서/기획서, 마케팅, 교육, HR, 영업, 데이터 분석, 법무, IT, 전략, 공공행정.
- **필터링**: 검색 기능 및 카테고리 탭을 통한 빠른 탐색.
- **복사 기능**: 개별 프롬프트 항목의 원클릭 클립보드 복사.

### 3.2. AI 프롬프트 인핸서 (AI Enhancer)
- **입력**: 사용자의 단순 텍스트 (예: "보고서 써줘")
- **출력**: 역할, 맥락, 제약사항이 포함된 정교한 프롬프트.
- **연동**: Google Gemini 1.5 Flash 모델 사용.

### 3.3. UX/UI 디자인 (Bento Style)
- 다크 테마 기반의 벤토 그리드(Bento Grid) 레이아웃.
- 반응형 웹 디자인 (Mobile/Desktop 최적화).
- 가독성 높은 폰트 사이즈 및 간격 적용.

## 4. 사용자 페르소나
- **기업 기획자/관리자**: 보고서 상신 및 분석 업무가 많은 직장인.
- **마케터/콘텐츠 크리에이터**: 매일 새로운 아이디어가 필요한 전문가.
- **개발자/IT 인력**: 기술 문서화 및 코드 리뷰 생산성을 높이려는 전문가.

## 5. 로드맵 (Future Roadmap)
- [ ] 사용자 커스텀 프롬프트 저장 기능 (Local Storage/DB)
- [ ] 프롬프트 활용 결과물 미리보기 기능.
- [ ] 팀 단위 프롬프트 공유 플랫폼 확장.
- [ ] 음성 입력(STT)을 통한 프롬프트 생성 지원.
