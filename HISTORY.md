# Change History - Prompt Architect

이 프로젝트의 모든 변경 사항은 이 문서에 기록됩니다.

## [v2.0.0] - 2026-05-09 (현재 버전)

### ✅ 추가된 기능
- **Bento Grid 테마 적용**: 현대적인 디자인 시스템으로 전체 UI 리뉴얼.
- **AI 프롬프트 인핸서**: Gemini API 연동을 통한 실시간 프롬프트 강화 기능 추가.
- **100선 라이브러리**: 10가지 직무 카테고리별 전문 프롬프트 데이터 구축.
- **다국어 지원**: 한국어와 영어 전환 기능 구현.
- **반응형 레이아웃**: 모바일에서도 가독성 있는 폰트 사이즈 및 카드 정렬 적용.

### 🛠 기술적 변경 사항
- **Tailwind CSS 4**: 최신 버전의 테일윈드 엔진으로 마이그레이션.
- **Vite Config**: 환경 변수 보안 주입 방식 최적화.
- **TypeScript**: 모든 데이터 구조에 정적 타입(Type/Interface) 적용.
